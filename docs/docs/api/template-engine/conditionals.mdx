# Conditionals

## Usage

<Tabs>

<TabItem value="block" label="Block">

```text
{{{? <condition> }}}
if output
{{{?? <else-if-condition> }}}
else if output
{{{??}}}
else output
{{{?}}}
```

</TabItem>

<TabItem value="inline" label="Inline">

```text
{{? <condition> }}if output{{?? <else-if-condition> }}else if output{{??}}else{{?}}
```

</TabItem>
</Tabs>
