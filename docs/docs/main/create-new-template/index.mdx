# Creating New Templates

This guide will walk you through the process of creating a new template from
scratch. You will learn things like:

-   [Where templates live and their structure](./templates.mdx)
-   [Packages](./packages.mdx)
-   [Dynamic files](./dynamic-files.mdx)
-   [Template Settings](./settings.mdx)
-   [Prompting](./prompts.mdx)

## Playground

Our [playground page](/playground) is a great resource to keep in mind as you go
through these docs. This page allows you to test features of templates quickly
and easily without touching code.

## Inspiration

If you're looking for some inspiration, you can check out our default template
code on
[GitHub](https://github.com/marcellino-ornelas/templates/tree/master/.tps). Our
templates always use best practices, advanced use cases, and incorporate new
upcoming features.
