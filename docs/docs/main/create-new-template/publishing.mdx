# Publishing

Publish your package to a dependancy manager of your choice and let others start
using your templates. Host your code on any version control system. Your
repository will be your template folder. Put all your templates packages inside
the repository folder.

```bash
| - some-template/ (repository)
	| - default
	| - <...other packages>
	| - settings.json
```

## How to prepare

### Prepare your exisiting template

If your template is in your local or global `tps` folder then move your template
out and into a place on your computer.

## version control system

You can host your template on any version control system.
