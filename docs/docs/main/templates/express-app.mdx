## Usage

```bash title="Usage"
tps express-app <app-name>

# or in a directory

tps express-app path/to/dir/<app-name>
```

```txt title="Creates"
| - <cmd-name>/
    | node_modules/
        | - all libraries node modules...
    | - src
        | - config/
            | - constrants.js
        | - middlewares/
            | - error-handler.js
        | - routes
            | - index.js
        | app.js
    | - .env
    | - package.json
    | - package-lock.json
```

## Copy

If you like this template, but want to modify a few things use the copy command.
It allows you to duplicate the template into your project and tailor it to your
needs.

```bash
# if your not initialized run
tps init

# copy template
tps copy express-app
```

### How to use

This template will generate a fully functioning express app. The name you pass
to the cli will be used as the apps directory name and the name in your
package.json. If you would like this app in a directory, then append the
directory path in front of the name.

example:

If you wanted to build a new express app for a new resturant called
`chickens-burgers` then you would run the following command:

```bash
tps express-app chickens-burgers

# if you want this directory in say a directory called projects, then run the following command
tps express-app ./project/chickens-burgers
```

this will create a new directory called `chickens-burgers` and create all the
nessessary files and directories you need for your new express app and install
the required packages using your package manager of choice.

```txt
|- funny-gifs/
    | node_modules/
        | - all libraries node modules...
    | - src
        | - config/
            | - constrants.js
        | - middlewares/
            | - error-handler.js
        | - routes
            | - index.js
        | app.js
    | - .env
    | - package.json
    | - package-lock.json
```

One your template is created you can `cd` into the directory and run the
following command to start the app

```bash
npm run start

# dev mode with nodemon
npm start:dev
```
